
# Flash notes



Tested flow:



1. Flash stock MIUI if returning from a broken state.

2. Boot/install working OrangeFox recovery with Goodix K6 touch support.

3. Flash Evolution X ZIP.

4. Format Data.

5. Wipe Cache / Dalvik.

6. Reboot System.

7. Do not flash Magisk recovery ZIP.



If root is needed, use Magisk APK to patch boot.img, then flash the patched boot image with fastboot.
